upNext

up-nxt.herokuapp.com

upNext is a web application that gives a nice presentation for software developed by programmers everywhere. The problem we are trying to solve is that we feel many developers are passed on during the job search process, one's github might be good but if the recruiter or HR person has no idea what they are looking at then there goes that opportunity. upNext solves that problem by allowing non technical recruiters or HR see a easy to understand visual representation of the software built.

This app was built using ReactJS/HTML5/CSS3/Jquery for front end and NodeJS/Express/PostgreSQL/Sequelize for the backend.
